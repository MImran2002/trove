# My Learning Journey

## Starting React Native

For react native, we need to use Expo as a framework to cretae the app we want. 

- To start we need to use the git command: 
`npx create-expo-app@latest`

- Continue to the link to set up the environment: https://docs.expo.dev/get-started/set-up-your-environment/?platform=android&device=simulated

- Choose an emulator, and download Andriod Studio. Follow each steps such as file path and such: /Users/imran/Library/Android/sdk
